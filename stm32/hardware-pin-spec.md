# 硬件标准规格和布线定义 — 四轮驱动闭环底盘控制节点

> 文档版本：V1.0（标准化硬件接口）  
> 适用系统：OpenClaw 底层运动学节点  
> 架构：OpenClaw（上层 AI 网关）+ STM32（底层实时控制器）

---

## 第一部分：核心硬件详细介绍

### 1. 主控单元：STM32F103RCT6 最小系统板（DevEBox）

| 参数 | 规格 |
|------|------|
| 核心架构 | ARM Cortex-M3 32位 RISC |
| 最大频率 | 72 MHz |
| 闪存 | 256 KB |
| SRAM | 48 KB |

**运动控制优势（本项目核心）：**

- **高级定时器 TIM1/TIM8**：支持死区插入和制动。本项目 TIM1 集中输出 4 路高频 PWM，实现电机转速平稳调节。
- **编码器接口**：硬件层面支持定时器直接解析 A/B 相正交编码器信号（TIM2/3/4/8）。MCU 可自动完成 4 个电机的 4 倍频高精度步数计数和速度测量，无需消耗 CPU 中断资源。
- **外设接口**：GPIO、USART（与 OpenClaw 通信）、I2C、SPI 等。

### 2. 动力驱动单元：四路闭环直流电机扩展板

- **驱动方案**：内置两个双通道全桥（H-bridge）电机驱动芯片（如 AT8236 或双 TB6612FNG），独立驱动 4 个直流减速电机。
- **接口标准化**：PH2.0-6P 标准端子（电机 A/B/C/D），电源线（M+/M-）与霍尔编码器电源（5V/3V3，GND）及信号线（A相/B相）集成为单一接口。
- **逻辑与电源隔离**：控制逻辑电平兼容 3.3V/5V；电机电源（Vin+ 蓝色端子）独立连接，防止反电动势和大电流导致 MCU 重启或损坏。
- **保护机制**：独立 STBY（使能/待机）控制引脚，紧急情况可硬件切断电源输出。

## 第二部分：标准引脚映射规范

> ⚠️ 严格按照下表进行电气连接。该映射方案优化了 STM32 硬件外设资源分配，确保无定时器冲突。

### 2.1 电源与全局控制

| 驱动板网络标签 | MCU 网络标识符 | 模式 | 功能描述 |
|:---|:---|:---|:---|
| GND | GND | 电源地 | **关键：两块板必须共地，确保逻辑电平参考一致** |
| STBY | 3V3 | 硬件高电平 | 驱动芯片激活（直连 3.3V，或连 GPIO 做软件控制） |
| Vin+ / Vin- | 电池组 | 外部电源 | **严禁连接 MCU**。建议 2S/3S 锂电池组 |

### 2.2 功率输出层（PWM 调速 + GPIO 转向）

> PWM 频率建议：**10kHz - 20kHz**（避免电机啸叫）  
> 方向引脚：通用推挽输出，无上下拉

| 电机通道 | PWM 调速引脚 | 定时器通道 | IN1（正反转1） | IN2（正反转2） |
|:---|:---|:---|:---|:---|
| Motor A（左前） | PWMA → **PA8** | TIM1_CH1 | AIN1 → **PC0** | AIN2 → **PC1** |
| Motor B（右前） | PWMB → **PA9** | TIM1_CH2 | BIN1 → **PC2** | BIN2 → **PC3** |
| Motor C（左后） | PWMC → **PA10** | TIM1_CH3 | CIN1 → **PC4** | CIN2 → **PC5** |
| Motor D（右后） | PWMD → **PA11** | TIM1_CH4 | DIN1 → **PB0** | DIN2 → **PB1** |

> ⚠️ **重要提醒**：PA9/PA10 默认为 USART1（串口1）引脚。本项目将其复用为 TIM1 PWM 输出，**通信串口必须重新映射至 USART2 或 USART3（PB10/PB11）**。

### 2.3 传感器反馈层（正交编码器读取）

| 参数 | 配置 |
|:---|:---|
| 定时器模式 | Encoder Mode (TI1 and TI2) |
| 输入捕获极性 | 上升沿（硬件自动 4 倍频） |
| 引脚配置 | 浮空输入或上拉输入 |

| 编码器通道 | A 相 | B 相 | 硬件编码器接口 |
|:---|:---|:---|:---|
| Motor A（左前） | E1A → **PA0** | E1B → **PA1** | TIM2_CH1 / TIM2_CH2 |
| Motor B（右前） | E2A → **PA6** | E2B → **PA7** | TIM3_CH1 / TIM3_CH2 |
| Motor C（左后） | E3A → **PB6** | E3B → **PB7** | TIM4_CH1 / TIM4_CH2 |
| Motor D（右后） | E4A → **PC6** | E4B → **PC7** | TIM8_CH1 / TIM8_CH2 |

## 附录：引脚汇总速查

```
PWM 输出：PA8  PA9  PA10 PA11  (TIM1_CH1~4)
方向控制：PC0~PC5, PB0, PB1      (GPIO 推挽输出)
编码器A：PA0  PA6  PB6  PC6    (TIM2/3/4/8 CH1)
编码器B：PA1  PA7  PB7  PC7    (TIM2/3/4/8 CH2)
通信串口：PB10 PB11             (USART3，替代 PA9/PA10 的 USART1)
```
